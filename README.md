# AUTOSAR_MCAL_LL_Nucleo-L476
Implement AUTOSAR Classic MCAL using Nucleo-L476 and Low-Layer library (LL)

Implementation includes:
- [DIO](MCAL/Dio/)
- [ADC](MCAL/Adc/)
- [SPI](MCAL/Spi/)
- [CAN](MCAL/Can/)
- [LIN](MCAL/Lin/)

Corresponding AUTOSAR documents can be found in [AUTOSAR_Doc](AUTOSAR_Doc)
